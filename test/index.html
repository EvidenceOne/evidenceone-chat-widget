Checklist de validação manual

  • ☐ Drawer 400px no desktop, 100% no mobile (<768px)
  • ☐ ESC fecha e retorna foco ao botão que abriu; backdrop click idem
  • ☐ Tab cicla apenas dentro do drawer; Shift+Tab faz wrap-around
  • ☐ Brand lock: alterar o rótulo/marca via DevTools no shadow root → autenticação recusa (ver test/lockdown.html)
  • ☐ Token lock: evidenceone-chat { --eo-green: red } no host page não muda nada interno
  • ☐ DevTools: Shadow DOM ativo; document.body sem scroll quando drawer aberto
  • ☐ Markdown nas respostas (negrito, listas, código, tabelas); XSS: <script>alert(1)</script> não dispara
  • ☐ Rede offline ao abrir: drawer mostra "Não foi possível conectar"
  • ☐ Fechar drawer durante streaming → fetch abortado no Network tab
  • ☐ Erro inline por mensagem: bloquear /partner/chat → bolha com borda vermelha e "!" clicável re-envia
  • ☐ 401 silencioso: re-auth automático e segunda tentativa funciona sem erro visível
  • ☐ A11y: drawer como dialog, message list como log; contraste WCAG AA
  • ☐ Perfil incompleto (422): estado "Cadastro incompleto" + evento eoBlocked no log
